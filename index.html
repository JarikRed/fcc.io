FCCID.link

What?

A simple search and URL shortener for FCC ID queries.

Try it here:
FCC ID:

Why?

I can never find the search form on the FCC site, so fccid.link should be easy enough to remember. Fccid.link provides a way to share FCC ID searchs with other people via links, email, IRC or IM.

How?

The URL scheme is simple:

  • https://fccid.link/"FCC ID"

Try these:

  • https://fccid.link/NPI
  • https://fccid.link/NPI71646
  • https://fccid.link/grantee/lego
